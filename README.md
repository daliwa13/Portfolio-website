# Iwona's Portfolio

Personal portfolio website built with pure HTML and CSS.

## Live Demo
- https://daliwa13.github.io/Portfolio-website/

## Tech Stack
- HTML5
- CSS3
- GitHub Pages
- Git and GitHub
- Chrome DevTools

## Overview
This project is a multi-page portfolio website created without frameworks, focused on clean structure, accessibility, and responsive layout.

## HTML & CSS Features Used

### HTML Elements and Structure
- Semantic layout elements: `header`, `nav`, `main`, `section`, and `footer`
- Multi-page navigation across Home, About, Work, and Contact pages
- Form and content sections organized for readability and usability

### CSS Possibilities Implemented
- **Animations** using `@keyframes` (e.g., color changes, moving/rotating SVG elements)
- **Transitions** for smooth visual state changes (e.g., hover and opacity/transform effects)
- **Transforms** like `scale`, `rotate`, and `translate` for interactive visuals
- **Responsive design** implemented with multiple `@media` breakpoints for desktop, tablet, and mobile layouts
- **Navigation pane type change** in responsive view: desktop uses a standard navigation list, while smaller screens switch to a hamburger-based dropdown menu
